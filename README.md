# Installation

### 1. Clone This Repo

```bash
git clone https://github.com/milanlizriadi/portfolio.git
```

### 2. Install the Packages

```bash
npm install
```

### 3. Setup TailwindCSS

```bash
npx tailwindcss -i ./src/input.css -o ./src/output.css --watch
```
